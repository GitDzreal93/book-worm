# 快速开始：LLM 大模型设置页

**功能分支**: `002-llm-settings`
**日期**: 2026-03-28

## 前置条件

- BookWorm 项目已运行（数据库已迁移、应用已启动）
- 至少有一个服务商的 API Key 可用

## 场景 1：首次配置 AI 服务

1. 打开 BookWorm，点击导航栏中的"设置"
2. 在设置页面中，选择一个服务商（如 OpenAI）
3. 在输入框中粘贴 API Key
4. 点击"验证"按钮，等待验证结果（应在 5 秒内返回）
5. 验证通过后，从模型列表中选择一个模型（默认为 gpt-4o-mini）
6. 点击"保存配置"
7. 返回书架，打开一本书，触发翻译功能，验证 AI 服务正常工作

## 场景 2：切换服务商

1. 进入设置页面
2. 选择新的服务商（如 Anthropic）
3. 输入该服务商的 API Key 并验证
4. 选择模型并保存
5. 后续 AI 功能将使用新服务商

## 场景 3：调整生成参数

1. 进入设置页面
2. 展开高级设置区域
3. 拖动温度滑块调整值（如从 1.0 调至 0.3）
4. 修改最大输出 Token 数
5. 保存并触发翻译，观察输出风格变化

## 场景 4：自定义提示词

1. 进入设置页面
2. 展开提示词配置区域
3. 修改翻译提示词（如添加"使用口语化表达"的要求）
4. 保存并触发翻译，确认输出格式符合自定义要求
5. 点击"恢复默认"按钮，确认提示词回到初始模板

## 环境变量兼容

如果用户未通过设置页配置 AI，系统将回退到环境变量：

```env
AI_PROVIDER="openai"
AI_MODEL="gpt-4o-mini"
OPENAI_API_KEY="sk-..."
ANTHROPIC_API_KEY="sk-ant-..."
GOOGLE_GENERATIVE_AI_API_KEY="AIza..."
```

设置页配置优先级高于环境变量。
